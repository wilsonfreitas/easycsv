 19  """ 
 20  easycsv.py 
 21   
 22  The easycsv module executes csv statements (a kind of csv DSL - domain  
 23  specific language) to insert/update/delete data into a database. 
 24   
 25  Easycsv was developed to be used with the Storm ORM framework, but it  
 26  could be easily adapted for others ORM frameworks (SQLAlchemy, SQLObject, ...). 
 27   
 28  >>> from storm.locals import * 
 29  >>> 
 30  >>> class Category(object): 
 31  ...     __storm_table__ = 'category' 
 32  ...     name = Unicode(primary=True) 
 33  ...     parent_name = Unicode() 
 34  ...     parent = Reference(parent_name, name) 
 35  ...  
 36  >>> 
 37  >>> database = create_database('sqlite:') 
 38  >>> store = Store(database) 
 39  >>> store.execute('CREATE TABLE category (name TEXT PRIMARY KEY, parent_name TEXT)') 
 40  <storm.databases.sqlite.SQLiteResult object at 0xa8a790> 
 41  >>>  
 42  >>> from easycsv import StormORM 
 43  >>>  
 44  >>> statements = '''\ 
 45  ... Category, Name, Parent 
 46  ... +, Expenses, 
 47  ... +, Internet, Expenses 
 48  ... ''' 
 49  >>>  
 50  >>> orm = StormORM(store=store) 
 51  >>> orm.execute(statements, modName='__main__') 
 52  (2, 0, 0, 2) 
 53   
 54  This session creates the table Category using the storm framework and inserts 
 55  two rows into the database.
